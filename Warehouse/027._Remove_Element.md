## 分析

原地去数，返回长度
也就只能弄得了这种暴力的……

```python
# 56 ms, faster than 39.80% 

class Solution(object):
	def removeElement(self, nums, val):
		for x in nums[:]:
			if x == val:
				nums.remove(val)
		return len(nums)

class Solution:
    def removeElement(self, nums, val):
        while val in nums:
            nums.remove(val)
        return len(nums)
```
