# 709. To Lower Case
> Difficulty = Easy

## 分析
大写转换为小写

```python
# 52ms, 22.82%

class Solution:
	def toLowerCase(self, str):
		"""
		:type str: str
		:rtype: str
		"""
		return str.lower()
```

接下来不用现成函数，用ASCII里**大写字母的十进制编码为65-90**，**小写字母的十进制编码为97-122**

```python
# 36ms, 63.87%

class Solution:
	def toLowerCase(self, str):
		"""
		:type str: str
		:rtype: str
		"""
		return ''.join([chr(ord(i)+32) if 64< ord(i) <91 else i for i in str])
```

## 总结
1. `ord()`函数就是用来返回单个字符的ascii值(0-255)
```python
>>> ord('d')
100
>>> ord('5')
53
```
2. `chr()`函数是输入一个整数(0-2550返回其对应的ascii符号
```python
>>> chr(100)
'd'
>>> chr(53)
'5'
```
3. `a.join(b)`函数，用`a`把`b`连接起来
''.join
```python
>>> a = 'like'
>>> b = ['do','you','me','you']
>>> c = 'IU'
>>> print(a.join(b))
'dolikeyoulikemelikeyou'
>>> print(a.join(c))
'IlikeU'
```
