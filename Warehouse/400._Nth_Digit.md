# 400. Nth Digit
> Difficulty = Easy

## 分析

> 题目要求
> 
> 连续的整数，找出第n位是几。

思路：

1-9有9个数，9×1位；10-99有90个数，90×2位；100-999有900个数，900×3位……

```python
# 32 ms, faster than 100.00%

class Solution:
    def findNthDigit(self, n: 'int') -> 'int':
        length = 1
        digit = 9
        start = 1
        while n > length*digit:
        	n -= length*digit
        	length += 1
        	digit *= 10
        	start *= 10
        start = start + (n-1)//length
        return int(str(start)[(n-1)%length])
```
