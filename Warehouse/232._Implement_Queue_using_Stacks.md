### 思路1：两个栈来模拟队列

参考了[$](https://blog.csdn.net/fuxuemingzhu/article/details/51345762)和[$$](https://github.com/apachecn/awesome-algorithm/blob/master/docs/Leetcode_Solutions/Python/232._implement_queue_using_stacks.md)的最后一种方法。

> 两个栈就是两个列表。当push时，向stack1里添加元素；当pop时，如果stack2不空，那么弹出其中元素就行；否则，如果stack1中有元素，需要先把stack1中的元素颠倒到stack2中。


> **时间复杂度？**

```python
# 32 ms, faster than 99.90%

class MyQueue:

    def __init__(self):
        """
        Initialize your data structure here.
        """
        self.stack1 = []
        self.stack2 = []

    def push(self, x):
        """
        Push element x to the back of queue.
        :type x: int
        :rtype: void
        """
        self.stack1.append(x)
        

    def pop(self):
        """
        Removes the element from in front of queue and returns that element.
        :rtype: int
        """
        if not self.stack2:
        	while self.stack1:
        		self.stack2.append(self.stack1.pop())
        return self.stack2.pop()
        

    def peek(self):
        """
        Get the front element.
        :rtype: int
        """
        if not self.stack2:
        	while self.stack1:
        		self.stack2.append(self.stack1.pop())
        return self.stack2[-1]
        

    def empty(self):
        """
        Returns whether the queue is empty.
        :rtype: bool
        """
        return not self.stack1 and not self.stack2
```

### 思路2：直接用列表(不合题意)

```python
# 32 ms, faster than 99.90%

class MyQueue:

    def __init__(self):
        """
        Initialize your data structure here.
        """
        self.lst = []
        

    def push(self, x):
        """
        Push element x to the back of queue.
        :type x: int
        :rtype: void
        """
        self.lst.append(x)
        

    def pop(self):
        """
        Removes the element from in front of queue and returns that element.
        :rtype: int
        """
        return self.lst.pop(0)
        

    def peek(self):
        """
        Get the front element.
        :rtype: int
        """
        return self.lst[0]
        

    def empty(self):
        """
        Returns whether the queue is empty.
        :rtype: bool
        """
        return not self.lst
```
