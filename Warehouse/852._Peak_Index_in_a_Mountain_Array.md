# 852. Peak Index in a Mountain Array
> Difficulty = Easy

## 分析

一序列是先增后减，让返回峰值所在位置

```python
# 64 ms, faster than 17.77%

class Solution:
	def peakIndexInMountainArray(self, A):
		"""
		:type A: List[int]
		:rtype: int
		"""
		return A.index(max(A))
```
