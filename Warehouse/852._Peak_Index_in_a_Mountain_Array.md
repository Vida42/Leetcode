## 二刷

实际题目是让练习二分查找……

```python
class Solution:
    def peakIndexInMountainArray(self, A: 'List[int]') -> 'int':
        left, right = 0, len(A)-1
        while left<right:
            mid = (left+right)//2
            if A[mid+1] > A[mid] > A[mid-1]:
                left = mid
            elif A[mid-1] > A[mid] > A[mid+1]:
                right = mid
            else:
                return mid
```

因为只有一个峰值，所以整个序列是先增后减。

所以如果处于增态，找右半；如果处于减态，找左半，否则返回。
