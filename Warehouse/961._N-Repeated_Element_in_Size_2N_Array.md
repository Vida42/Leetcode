# 961. N-Repeated Element in Size 2N Array
> Difficulty = Easy

## 分析
2N长度的列表，N+1个不同元素，其中有一个元素重复了N次，让找出那个数。

```python
# 84ms, 46.28%

class Solution:
    def repeatedNTimes(self, A):
        """
        :type A: List[int]
        :rtype: int
        """
        for i in A:
        	if A.count(i) > 1:
        		return i
```

## 总结
1. 使用`list.count()`，可以得到每一个元素，在list中出现的次数
```python
>>> b = ['do','you','me','you']
>>> z = b.count(b[1])
>>> print(z)
2
```
